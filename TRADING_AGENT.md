# SOXL Swing Trading Agent

## Strategy Summary
- **Ticker**: SOXL (3x leveraged semiconductor ETF)
- **Signal**: EMA(5) / EMA(20) crossover + Fear & Greed Index momentum gate + velocity gate
- **Exit**: 11% trailing stop from peak, OR EMA crossover sell signal
- **Backtest**: ~5909% return, Sharpe 2.83, -34% max drawdown (Oct 2023–Jun 2026)

## Your job each hour

1. Run `python trade.py` and read the JSON output.
2. Use the Robinhood MCP tools to check current portfolio state.
3. Execute the trade if action is BUY or SELL.
4. If action is HOLD, do nothing.

## Step-by-step

### Step 1 — Get the signal
```bash
cd /Users/vikramatmuri/Downloads/Projects/robinhood_trader
.venv/bin/python trade.py
```
Read the `action` field in the JSON output.

### Step 2 — Check current Robinhood state
Use `mcp__robinhood__get_equity_positions` to see if you currently hold SOXL.
Use `mcp__robinhood__get_portfolio` to see available cash.

### Step 3 — Execute if needed

**If action == "BUY"** and you have NO current SOXL position:
- Use `mcp__robinhood__get_equity_quotes` for SOXL to get current ask price
- Use `mcp__robinhood__place_equity_order` with:
  - symbol: "SOXL"
  - side: "buy"
  - type: "market"
  - quantity: floor(available_cash / current_price)  ← use ALL available cash
  - time_in_force: "gfd"

**If action starts with "SELL"** and you DO hold SOXL:
- Use `mcp__robinhood__get_equity_positions` to get exact share count
- Use `mcp__robinhood__place_equity_order` with:
  - symbol: "SOXL"
  - side: "sell"
  - type: "market"
  - quantity: your full share count
  - time_in_force: "gfd"

**If action == "HOLD"**: Do nothing. Log the current state.

### Step 4 — Confirm
After placing an order, use `mcp__robinhood__get_equity_orders` to verify it was filled.

## Important rules
- NEVER trade outside market hours (9:30am–4:00pm ET, Mon–Fri)
- Trade.py already enforces this — if it prints "Market is closed", stop immediately
- Always use market orders for guaranteed fills on SOXL (highly liquid ETF)
- Use ALL available cash on a BUY signal (100% position sizing)
- Always sell the FULL position on a SELL signal
- The trailing stop is tracked in `trade_state.json` — do not edit this file manually
